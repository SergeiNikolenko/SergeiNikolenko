Проекты

Генерация иммунных пептидов

Данный проект автоматизирует проектирование потенциально иммуногенных пептидов из структур белка. Скрипт определяет сайты расщепления, генерирует кандидаты, оптимизирует их иммуногенность и выбирает лучшие из них для отображения.

Стек: Python, AlphaFold2, MDTraj, NumPy, BioPython, Skikit-Learn

Кластеризация антител

Этот скрипт предназначен для извлечения последовательностей аминокислотных цепей антител из файлов PDB. Для обработки файлов используется база данных SAbDab. Он позволяет извлекать аминокислотные последовательности, удалять пустые файлы и кластеризовать последовательности с помощью различных алгоритмов.

Стек: Python, Anaconda, DBSCAN, K-means, Hierarchical Clustering, Biopython, Scikit-learn, Numpy, Matplotlib, Scipy, Seaborn
