Immune Peptide Generation

The Immune Peptide Generation project automates the design of potentially immunogenic peptides from protein structures. It identifies cleavage sites, generates candidate peptides, optimizes their immunogenicity, and selects the best ones for display.

This project uses a combination of Python scripts and tools like AlphaFold2, OpenBabel, and AutodockVina to create an efficient and user-friendly solution for designing peptides for potential use in vaccine development and immunotherapy.

By providing an automated pipeline for peptide generation, this project aims to save time and resources for researchers working in the field of immunology and drug discovery. The script can be easily customized and integrated into various research workflows.
